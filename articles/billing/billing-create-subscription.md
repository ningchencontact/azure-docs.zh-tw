# <a name="create-an-additional-subscription-in-the-azure-portal"></a>在 Azure 入口網站中建立額外的訂用帳戶

您可以在 Azure 中為帳戶建立額外的訂用帳戶。 您可能需要一個額外的帳戶以避免達到訂用帳戶限制、基於帳單與安全性目的建立個別環境，或基於合規性因素隔離資料。

如果您想要在組織的 Enterprise 合約下方建立 Azure 訂用帳戶，您必須具備適用於組織的帳戶擁有者角色。

如果您還沒有 Azure 訂用帳戶，請建立[免費帳戶](https://azure.microsoft.com/free)。

## <a name="create-an-additional-azure-subscription"></a>建立額外的 Azure 訂用帳戶

1. 登入 [Azure 入口網站](https://portal.azure.com)。
1. 尋找**訂用帳戶**。

   ![顯示在入口網站中搜尋訂用帳戶的螢幕擷取畫面](./media/billing-create-subscription/billing-search-subscription-portal.png)

1. 選取 [新增] 。

   ![在 [訂用帳戶] 檢視中顯示 [新增] 按鈕的螢幕擷取畫面](./media/billing-create-subscription/subscription-add.png)

您也可以利用程式設計方式來建立額外的訂用帳戶。 如需詳細資訊，請參閱[以程式設計方式建立 Azure 企業版訂用帳戶](../azure-resource-manager/programmatically-create-subscription.md)。
