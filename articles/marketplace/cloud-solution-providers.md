# <a name="cloud-solution-providers"></a>雲端解決方案提供者

軟體供應項目可以立即觸及數百萬名符合資格，幾乎不需要的雲端解決方案提供者 (CSP) 合作夥伴所提供的 Microsoft 客戶行銷和銷售的投資。  發行者可以立即利用 CSP 合作夥伴通路，除了布所提供的公開發行[Microsoft 的網路店面](https://docs.microsoft.com/azure/marketplace/comparing-appsource-azure-marketplace)。

發行者可以選擇以加入供應項目在 CSP 合作夥伴通道中，新的供應項目或一個現有的可用性。 此選項可讓 CSP 合作夥伴銷售您的產品，並為客戶建立搭售的方案。

發行者負責為客戶提供協助修正支援，以及提供 CSP 合作夥伴的機制和/或客戶與您連絡支援。 它是提供訓練、 使用者文件中，CSP 合作夥伴和服務 （視情況） 的健全狀況/中斷通知，讓 CSP 合作夥伴都有能力處理層 1 的支援要求，來自客戶的最佳作法。

下表顯示供應項目類型，可以選擇在 CSP 合作夥伴通路的計價模式：

| **供應項目類型**    | **BYOL**  |  **免費** | **PAYG**   |
| :---------------- | :---------|:----------|:-----------|
| VM  | 是 | 是 | 是 |
| 容器 | N/A | 是 | N/A |
| 解決方案範本 | N/A | 是 | N/A |
| 受控應用程式 | N/A | 是 | 是 |
| IT 解決方案 SaaS （每個站台定價） | N/A | 是 | 是 |
|   |   |   |

## <a name="how-to-opt-in"></a>如何選擇加入

Marketplace 或 Storefront 索引標籤在 Cloud Partner 入口網站中的供應項目建立體驗中找到的 CSP 合作夥伴通道選擇加入設定。
