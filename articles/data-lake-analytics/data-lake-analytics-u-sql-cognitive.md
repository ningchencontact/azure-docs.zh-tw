# <a name="get-started-with-the-cognitive-capabilities-of-u-sql"></a>開始使用 U-SQL 的辨識功能

## <a name="overview"></a>總覽
U-SQL 的辨識功能讓開發人員可以在其公司的巨量資料程式中使用 put 智慧。 

下列使用認知功能的範例可供使用：
* L[偵測臉部](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* L[偵測表情](https://github.com/Azure-Samples/usql-cognitive-imaging-emotion-detection-hello-world)
* L[偵測物件（標記）](https://github.com/Azure-Samples/usql-cognitive-imaging-object-tagging-hello-world)
* L[OCR (光學字元辨識)](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* 文字[關鍵片語擷取 & 情感分析](https://github.com/Azure-Samples/usql-cognitive-text-hello-world)

## <a name="registering-cognitive-extensions-in-u-sql"></a>在 U-SQL 中註冊認知擴充功能
開始之前，請遵循本文中的步驟，在 [U-SQL] 中註冊認知延伸模組：[在 sql-dmo 中註冊認知延伸](/u-sql/objects-and-extensions/cognitive-capabilities-in#registeringExtensions)模組。
