| **白皮書** | **說明** |

| [Azure Cosmos DB 和個人資料](https://servicetrust.microsoft.com/ViewPage/TrustDocuments?command=Download&downloadType=Document&downloadId=87cc6456-4b23-473c-94d3-6c713b8b8956&docTab=6d000410-c9e9-11e7-9a91-892aae8839ad_FAQ_and_White_Papers)| 本文會為管理雲端式資料庫和 (或) 內部部署資料庫的 Azure Cosmos DB 客戶提供指引，也會為需要確保其資料庫系統中的個人資料處理及保護，是否符合目前規則的人員提供指引。 |
