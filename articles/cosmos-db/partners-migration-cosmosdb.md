# <a name="azure-cosmos-db-nosql-migration-and-application-development-partners"></a>Azure Cosmos DB NoSQL 移轉和應用程式開發合作夥伴

|![Striim][2]    |  從各種資料來源[將即時資料移至 Azure Cosmos DB](https://www.striim.com/striim-for-azure-cosmos-db/)。 <br/> Striim 讓您可以更輕易地從各種內部部署來源 (包括企業文件和關聯式資料庫、感應器和記錄檔) 將資料即時收集並移動到 Azure Cosmos DB 中。 |   USA   |  [網站](https://www.striim.com/)       |
