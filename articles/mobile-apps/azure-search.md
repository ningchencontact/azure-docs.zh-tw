# <a name="ai-powered-cloud-service-with-azure-search"></a>具備 Azure 搜尋服務的 AI 技術雲端服務
[Azure 搜尋服務](https://azure.microsoft.com/services/search/)是一個搜尋即服務雲端解決方案，可為開發人員提供 api 和工具，以在 web、行動和企業應用程式中，為私人、異類內容新增豐富的搜尋體驗。 您的自訂程式碼會叫用資料擷取 (索引編制) 來建立和載入索引。 另一方面，應用程式的程式碼會發出查詢要求和處理回應。 搜尋體驗會透過 Azure 搜尋服務的功能，在您的用戶端中進行定義，而查詢會針對您在服務上建立、擁有及儲存的持續性索引執行。

## <a name="azure-search-features"></a>Azure 搜尋服務功能
- 使用全文檢索搜尋、簡單查詢語法和 lucene 查詢語法的**自由格式文字搜尋**。
- **簡單評分設定檔**，可將相關性模型化為檔本身中的值函數。
- **地理搜尋**可讓使用者根據搜尋結果與實體位置的鄰近程度來流覽資料。
- **使用者體驗**具有自動完成的功能、搜尋建議、排序和分頁。
- 適用于影像和文字分析的**認知搜尋**可以套用至索引管線，以從原始內容中解壓縮文字資訊。
- 儲存在索引編制期間建立之擴充的**知識存放區**。
- 連線至 Azure SQL Database、Azure Cosmos DB 或 Azure Blob 儲存體作為**資料來源**，以在主要資料存放區中將可搜尋的內容解壓縮。
- 您可以使用 [匯**入資料] wizard**來設定索引**器、索引設計**工具來建立索引，以及**搜尋 explorer**來測試查詢及精簡評分設定檔。

尋找 Azure 搜尋服務[功能](/azure/search/search-what-is-azure-search#feature-descriptions)的詳細描述。
