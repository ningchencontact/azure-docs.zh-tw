# <a name="scalability-targets-for-premium-block-blob-storage-accounts"></a>Premium 區塊 blob 儲存體帳戶的擴充性目標

Premium-效能區塊 blob 儲存體帳戶已針對使用較小 kb 範圍物件的應用程式優化。 這適用于需要高交易率或一致低延遲儲存體的應用程式。 高階效能區塊 blob 儲存體是設計來隨著您的應用程式調整。 如果您的案例需要部署的應用程式需要每秒數百個要求或 pb 的儲存容量，請在[Azure 入口網站](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade)中提交支援要求來聯絡 Microsoft。
