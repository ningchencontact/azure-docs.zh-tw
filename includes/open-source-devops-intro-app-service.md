[Azure App Service](/azure/app-service/overview) 可讓您建置及裝載程式碼。 此程式碼的形式可以是 Web 應用程式、行動後端和 RESTful API。 透過 App Service，您可以使用所選擇的程式設計語言來開發程式碼，而不需要管理基礎結構。 App Service 同時支援 Windows 和 Linux。 支援從任何 Git 存放庫進行自動化部署，包括 GitHub 和 Azure DevOps。
