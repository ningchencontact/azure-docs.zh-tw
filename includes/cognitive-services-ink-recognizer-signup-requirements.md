您必須有具備筆跡辨識器 API 存取權的[認知服務 API 帳戶](../articles/cognitive-services/cognitive-services-apis-create-account.md)。 如果您沒有 Azure 訂用帳戶，可以[建立免費帳戶](https://azure.microsoft.com/try/cognitive-services/)。 您可以在建立帳戶後從 [Azure 入口網站](../articles/cognitive-services/cognitive-services-apis-create-account.md#get-the-keys-for-your-resource)取得訂用帳戶金鑰，也可以在啟用免費試用版後從 [Azure 網站](https://azure.microsoft.com/try/cognitive-services/my-apis)來取得。
