1. 從 Visual Studio 功能表中，選取 [檔案] > [新增] > [專案]。
2. 在 [新增專案] 對話方塊的 [範本] 區段下，選取 [Visual C#] > [Web]。
3. 選取 **ASP.NET Core Web 應用程式**。
4. 指定新應用程式的名稱 (或使用預設值)，然後選取 [確定] 。
5. 選取 **Web 應用程式**。
6. 請勾選 [啟用 Docker 支援] 核取方塊。
7. 選取您想要 (Windows 或 Linux) 的容器類型，，然後按一下 [確定]。
