**問**：為什麼要變更我現有閘道的位置？ <br/>
**答**：這些閘道建立之前 2017 5 月 3 日已移至 Azure AD 租用戶的原始位置的工作或學校帳戶建立的閘道資源。 不過，這些變更應該不會影響目前執行的邏輯應用程式，而應該是繼續如往常般運作。 隨著閘道在 5 月公開上市，閘道資源位置可能會與邏輯應用程式的位置有所不同。
