# <a name="sql-database-documentation"></a>SQL Database 文件

Azure SQL Database 是一般用途關聯式資料庫即服務 (DBaaS)，它是根據最新穩定版本的 Microsoft SQL Server 資料庫引擎而建置。 SQL Database 是高效能、可靠且安全的雲端資料庫，您可以使用選定的程式設計語言來建置資料驅動的應用程式和網站，無需管理基礎結構。 了解如何透過我們的快速入門、教學課程和範例使用 SQL Database。

## <a name="5-minute-quickstarts"></a>5 分鐘快速入門

Azure SQL Database 可讓您使用 Azure 入口網站、Azure CLI 和 Azure PowerShell，輕鬆地執行基本管理工作。 深入了解如何使用下列快速入門範例來執行基本管理工作：

- [Azure 入口網站](/azure/sql-database/sql-database-single-database-get-started)
- [Azure PowerShell](/azure/sql-database/sql-database-get-started-powershell)
- [Azure CLI](/azure/sql-database/sql-database-get-started-cli)

## <a name="step-by-step-tutorials"></a>逐步教學課程

了解如何使用 SQL DB 建立、管理和移轉資料庫。

- 使用我們的快速入門指南開始使用[單一資料庫](/azure/sql-database/sql-database-single-database-quickstart-guide)

- 使用 [Data Migration Assistant](/azure/dms/tutorial-sql-server-to-azure-sql)，透過[匯入 BACPAC 檔案](/azure/sql-database/sql-database-import)將現有的資料庫遷移至 Azure
